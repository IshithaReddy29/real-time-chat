**Real-Time Chat Application**

A full-stack real-time chat application built using React, Node.js, Express.js, MongoDB and Socket.io.

**Features**

- Real-time messaging
- MongoDB message storage
- Username support
- Message timestamps
- Persistent chat history
- Socket.io integration

**Tech Stack**

- React
- Node.js
- Express.js
- MongoDB
- Socket.io
